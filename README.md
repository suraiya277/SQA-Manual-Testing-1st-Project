# SQA Manual Testing Project

## Overview
This repository contains a **comprehensive Manual Testing project** for a demo **E-commerce Web Application(https://sauce-demo.myshopify.com/)**.  
It demonstrates the **complete software testing lifecycle**, from planning and test case design to defect reporting and requirement traceability.

---

## Application Type
**E-commerce Web Application**  

---

## Testing Scope
- **Functional Testing:** Validate all functional requirements  
- **Smoke Testing:** Verify critical workflows for stability  
- **Regression Testing:** Ensure changes do not impact existing functionality  

---

## Modules Tested
- Signup  
- Login  
- Cart
- Orders  
- Payment  

---

## Artifacts Included

| Artifact | Description | File Location |
|----------|-------------|---------------|
| **Test Plan** | Detailed test strategy, objectives, and scope | `Test-Plan/Test Plan-1st-manual-project.pdf` |
| **Test Cases** | Structured scenarios with step-by-step execution | `Test-Cases/sauce-demo-1st-manual_project-Test case.xlsx` |
| **Bug Report** | Documented defects with severity, priority | `Bug-Reports/Bug_Report_ProjectName.xlsx` |
| **Requirement Traceability Matrix (RTM)** | Mapping of requirements to test cases to ensure full coverage | `RTM/RTM-sauce-demo-1st-manual_project.xlsx` |
| **Test Report** | Summary of executed tests, pass/fail metrics, and observations | `Test-Report/Test report-sauce-demo-1st-manual_project.docx` |

---

## Tools Used
- **MS Excel:** Test case design, bug reporting, RTM
- **MS Word:** Test Plan and Test Report documentation
- **Jira:** Bug tracking and issue management using Kanban board
- **Manual Testing:** Functional, smoke, and regression testing
- **Version Control:** Git & GitHub
